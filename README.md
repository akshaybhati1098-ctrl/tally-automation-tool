# Tally Excel to XML Converter (Web Edition) with PDF/Image OCR

Convert Excel invoice files to Tally‑compatible XML.  
Also convert PDF or image bills to Excel using OCR with company‑specific extraction rules.

## Features

- **Excel → XML** – Generate Tally XML from Excel (uses your existing `convert_menu.py`).
- **PDF/Image → Excel** – Extract invoice data from scanned bills using Tesseract OCR.
- **Company Rules Manager** – Define custom extraction keywords for different companies.
- **Mapping Editor** – Edit GST rate → ledger mappings.
- **Settings** – Theme, default voucher type, etc. (saved in browser).
